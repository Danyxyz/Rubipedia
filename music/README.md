# Background Music

To add background music to your Rubipedia site:

1. Add your MP3 file to this folder
2. Rename it to `background.mp3` OR
3. Update the audio source in `index.html` to match your filename:
   ```html
   <source src="music/your-filename.mp3" type="audio/mpeg">
   ```

## Recommended Music

For the Egyptian mystical theme, consider:
- Ambient Egyptian music
- Meditation/spiritual frequencies (432Hz, 528Hz)
- Binaural beats
- Lofi mystical beats
- Ancient temple ambience

The player features:
- ♫ Play/Pause controls
- Volume slider
- Progress bar with time display
- Looping playback
- Draggable player window
- Collapsible interface

**Note:** Most browsers block autoplay with sound. Users will need to click the play button to start the music.
